=====
konch
=====

Customizes your Python shell.

Get it now
----------

.. code-block:: bash

    $ pip install konch


Get started
-----------

Just use the ``konch`` command to run your interactive Python interpreter.

.. code-block:: bash

    $ konch init
    $ konch


Requirements
------------

- Python 2 or 3 (tested on 2.7 and 3.3)
